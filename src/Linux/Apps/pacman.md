# pacman

## Кеш

Уделение всех версий из кеша кроме последних
~~~~
paccache -rk1
~~~~

Удаление всех версий из кеша которых нет в системе
~~~~
paccache -ruk0
~~~~

Удалить весь кеш из системы
~~~~
pacman -Scc
~~~~

Поиск файлов внутри пакета
~~~~
pacman -Ql firefox | grep "<some file name>"
~~~~
